# Contributing

Thank you for your interest in dBconverter.

## Bug reports

Bug reports are welcome through GitHub Issues. Please use the bug report form
and include:

- the steps needed to reproduce the problem
- the expected and actual behavior
- the browser, operating system, and device
- screenshots when they help explain the problem

Please do not include passwords, credentials, personal information, or other
sensitive data.

## Pull requests

External pull requests are not accepted. Development is maintained in a
separate private repository.

Issue reports may still be used to identify and prioritize fixes.
